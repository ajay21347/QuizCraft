# PDF to Quiz

An AI-powered web application that converts PDF documents into interactive quizzes.

## Features

- Upload any PDF
- Extract questions and answers automatically
- Detect missing answers
- Generate missing answers using Groq AI
- Edit extracted questions before quiz generation
- Store questions in MongoDB
- Generate timed quizzes
- Review answers with explanations
- Export questions to CSV, Excel, and JSON

---

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- Axios

### Backend

- Python
- FastAPI
- PyMuPDF
- pdfplumber
- Groq API
- MongoDB
